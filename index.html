Rules:
• Type what you see on the top, and then hit submit to move onto the next thing to type
• You can push reset to delete everything in the textbox
• You can also use backspace to delete what you've typed, but you can't hold it down
• Everything you have to type is in lowercase, so you don't need to worry about that
• This is just an ordinary, run of the mill, typing game, so relax and have fun!
• Just in case you forget what your keyboard looks like, I've put a helpful little guide below the textbox
• Use the keyboard guide if you get stuck!! Remembering where the keys are can be pretty hard sometimes
• Have fun :)
